Preparation: Energy Balance

Estimated time to complete: 2.5 hours.

Learning Objectives and Questions Guide

  • You will be able to describe the strengths and limitations of evaluating body weight using the following methods and be able to make appropriate recommendations using that information.
  • You will be able to describe the energy balance principle and how to measure/estimate energy intake and energy output.
  • You will be able to describe genetic factors which influence your dietary intake.

The preparation guide lists questions you should be able to answer and key terms for the objectives. It will be very useful as a reference in the timed lesson quiz. (Click to download guide)
