Preparation: Proteins and Amino Acids

Estimated time to complete: 2.5 hours.

Learning Objectives and Questions Guide

  • You will be able to describe the structure of amino acids and protein.
  • You will be able to describe the digestion, absorption, and transport of dietary protein.
  • You will be able to describe the synthesis of proteins in the body and their functions. 
  • You will be able to describe the role of protein in health and disease.
  • You will be able to describe the rationale and calculate the protein recommendations for an individual.
  • You will be able to describe possible benefits and limitations associated with plant-based dietary plans.

The preparation guide lists questions you should be able to answer and key terms for the objectives. It will be very useful as a reference in the timed lesson quiz. (Click to download guide)
