Course-Specific Setup for Instructor:

For this course we will be having the students download MySQL. You need to download both MySQL Server and MySQL Workbench for this class. This is available on multiple operating systems for free. Please take some time to familarize yourself with these tools so that you will be able to answer student questions. 

 

At the start of week 2, the students will need to be assigned into groups to last for the remainder of the semester. After you have created the groups, post the group information on the "groups" page found within the "course" folder.

 

Here is a sample email you can send out the first week of class. 

"Welcome to CIT 111! I'm excited to experience this course with you. We're going to hit the ground running this first week - Intro Week. Be sure and post your picture to the Personal Information in the "Icebreaker Discussion Board". Click http://[stream.byui.edu] /tutorials/brainhoney/New/Upload_Photo.swf for a 2-minute walkthrough on how to do this.I'll see you in the Icebreaker Discussion Board."
