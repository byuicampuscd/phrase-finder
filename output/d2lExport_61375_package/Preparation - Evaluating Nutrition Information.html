Preparation: Evaluating Nutrition Information

Estimated time to complete: 2 hours.

Learning Objectives and Questions Guide

  • You will be able to describe the process of the scientific method.
  • You will be able to describe criteria for determining the "strength of the evidence".
  • You will be able to distinguish between reliable and unreliable nutrition information.

The preparation guide lists questions you should be able to answer and key terms for the objectives. It will be very useful as a reference in the timed lesson quiz. (Click to download guide)
